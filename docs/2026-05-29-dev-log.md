# 开发记录

## 2026-05-29

### 已完成
- 安装器自动从 `C:\Program Files (x86)\ControlCenter\InsydeDCHU.dll` 复制驱动文件到安装目录。
- 添加了实验性功能入口，分区测试工具随安装包发布。
- 应用场景配置已从自动化页拆分为独立设置方向，默认按应用图标取色。
- 设置页已改为左侧导航 + 右侧独立页面，避免顶部标签页和固定窄宽度造成漏字。
- 自动化页和应用场景页已拆开，应用场景编辑器放宽控件尺寸并去掉泛化的“添加”入口。
- 应用场景前台检测由托盘进程定期写入 `foreground-app.json`，服务读取该状态并按活动窗口应用规则。
- 自动图标取色开启时不再被隐藏的手动颜色控件覆盖，应用图标色会作为固定颜色或单色呼吸的目标色。
- 添加敲字闪烁：托盘进程用低级键盘 Hook 记录输入时间，服务端按状态文件临时抬高亮度并平滑回落。
- 音乐模式拆分为“按电平变色”和“仅亮度脉冲”，亮度脉冲不再改变颜色。
- 音乐模式加入噪声门、节拍阈值和峰值保持，颜色变化改用平滑包络，减少全频电平造成的快速乱跳。
- 音乐模式 UI 改为预设优先：普通用户可直接选择流行、摇滚、经典、电子。
- 噪声门、节拍阈值、响应速度、衰减速度等参数移入“显示高级参数”。
- 支持最多 8 个命名自定义音乐预设，可从当前高级参数保存、覆盖和删除。
- 设置页新增“诊断”页，集中显示服务状态、驱动 DLL、当前前台应用、命中的应用场景、更新检查和配置目录。
- 修复高 DPI / 字体缩放下设置页缺字和按钮文字被切的问题：统一放大行高、按钮高度、滑条行高，复选框改为自适应尺寸。
- 重新调整应用场景编辑器布局，避免“选择进程”等按钮文字被截断。
- 增加内置备用 `InsydeDCHU.dll`：安装包会先尝试使用 `ControlCenter` 中的驱动 DLL，找不到时保留随安装包发布的备用 DLL。
- 新增 `ColorfulLedKeyboard.Tests` 测试项目，覆盖 HSV 色轮、RGB 循环重建连续性、应用场景颜色、音乐预设去重限制、运行态克隆等纯逻辑。
- 修复若干边界问题：下拉框索引读取改为安全夹取，应用场景空列表刷新时明确清空选择。

### 当前问题

- 仍需在测试机验证应用场景是否能随 Spotify 等活动窗口即时切换。
- 分区功能不再进入主线，只保留实验性测试。
